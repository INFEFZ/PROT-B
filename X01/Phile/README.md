# Projektarbeit-Programmiertechnik-B


Unter Codes sind 3 Ordner mit Codes darin.
- Der Ordner Code ist der originale Code, der auch in der Dokumentation erwähnt wird.

- Der Ordner Code+ ist der originale Code mit angefangenen Zusatzfunktionen und verbesserungen.

- Der Ordner Code+_ultimately_AI, zeigt was es mit KI möglich ist und wie der Code aufgebaut werden kann, wenn er von der Grundlage des Originalcodes ausgeht.
